# CASE_PAGE_SPEC.md
BiBi 症例ページ 生成仕様（Gold Standard）

## 0. 目的（固定）
- 症例ページを量産するのではなく、BiBiクオリティを維持した「下書き」を自動生成する。
- AIは下書き生成係、人が最終判断と責任。

## 1. 出力物（最低ライン）
- WordPress 症例HTML（本仕様の主成果物）
- note / アメブロ：本文はWPをベースに媒体最適化（冗長なCSSなし）
- Instagram：①スライド構成案（8枚）②キャプション

---

## 2. ページ全体の設計思想（絶対）
- 結論ファースト（B/A画像 or 変化要約 → 1文結論 → 施術概要）
- 「線の悩み」ではなく「構造（面）・光と影・バランス」で説明する
- SEO/医療広告/ブランドトーンを同時に満たす（煽りすぎない、でも弱くしない）
- 回遊設計：総合ページへの導線 + 関連施術への導線
- スマホ最適化：レスポンシブ前提（横スクロールはテーブルのみ許容）

---

## 3. セクション順（固定）
### S1. 冒頭画像（2枚推奨）
- grid（auto-fit）で2枚を横並び（スマホは縦積み）
- altは「年代 性別 施術名 症例写真〔角度B/A〕｜大阪 心斎橋BiBiクリニック」で統一

### S2. ファーストビュー結論（1行）
- 形式：▶ ○○（年代）の△△を改善し、□□で若々しい印象へ
- 18px程度、背景色付き、左ボーダーで強調

### S3. 施術概要ボックス（早い位置）
必須項目：
- 施術内容
- リスク・副作用（主要 + 重大：塞栓は必ず含める）
- 使用製剤（製剤名 + 本数/単位 ※cc表記はしない）
- 価格（モニター/通常のどちらか。ルールはPRICE_SPECに従う）
- 回数
- 施術時間
※価格表記がある場合、免責文を必ず末尾に追加：
「＊本ページ記載のモニター制度・料金は投稿時点の内容です。今後、内容が変更・終了となる場合があります。」

### S4. 患者様プロフィールとお悩み（2カラム）
- 左：補助B/A（1枚）
- 右：プロフィール → 主なお悩み → 年代背景（一般論） → 当院方針
- 末尾に総合ページ導線（例：/service/hyaluronic-acid/lift/）
※ユーザー向け表記で「ピラー」は使用禁止。「総合ページ」と記載。

### S5. 施術内容・使用製剤（テーブル + 解説）
- テーブル（スマホで横スクロール可）
- 行：使用製剤 / 注入部位 / （必要なら）セッション分け
- 直下に「製剤選定理由」：製剤の役割を“層・目的”で語る
- 製剤比較コラム等へ内部リンク（サイトマップの正規スラッグのみ使用）

### S6. Dr.のこだわりポイント（3ブロック固定）
1) 設計：線ではなく面、構造（骨・脂肪・靭帯）を評価
2) 美的基準：光と影、ハイライト位置、立体感
3) 安全性：塞栓リスク、手技の安全対策、承認製剤、緊急体制
- 途中に「▶ BiBiでは支える構造から設計する」系の太字まとめを1つ挿入
- 右カラムに施術風景/手元/院長写真など（任意）

### S7. Before / After（複数角度）
- 前置き：見るポイント（例：ほうれい線/頬の高さ/フェイスライン/ボリューム）
- 画像：正面＋左右斜め（3枚推奨）
- 末尾にリスク/製剤/価格/回数/効果個人差の注意を小さく記載
※価格が出る場合は免責文もここかS3に必ず入れる（重複OK）

### S8. 施術後の変化（院長コメント）
- 左：院長写真 + 表記「院長 山本 幸一郎」
- 右：コメントボックス
文体ルール：
- 一人称「私」で語る（例：「私が意識したのは…」）
- “自然/構造/光/表情の動き”を必ず入れる
- 患者様の言葉（あれば）を1文引用（過度な誇張は禁止）

### S9. 患者様の感想（引用 + 解説）
- 引用はblockquote、強調は部分色替え可
- 直下で「この年代の課題→当院アプローチ」の一般化で締める

### S10. この症例が向いている人（チェックリスト）
- 3項目が基本（年代/悩み/希望）
- “自己一致”を起こす言葉にする（煽りは避ける）

### S11. 関連ページ導線（2段構成）
- (A) 総合ページへ（例：ヒアルロン酸リフト総合ページ）
- (B) 併用/関連施術へ（例：下顔面/顎、ボトックス 等）
※サイトマップの正規スラッグのみ。推測スラッグ禁止。

### S12. フッター（簡易）
- 著作権表記など（WP側で共通なら省略可）

---

## 4. 禁止事項 / 注意
- cc（注入量）を具体数値で書かない（必要なら「本数」「単位」まで）
- 煽り表現の禁止（断定・誇張：絶対/必ず/誰でも等は避ける）
- 医療広告上のNG：ビフォーアフターで過剰に効果保証しない、個人差明記
- 内部リンクはサイトマップの正規URLスラッグに厳密準拠
- 「ピラー」という用語をユーザー向け文章で使わない（総合ページに統一）
